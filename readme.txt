# art2xmb - PS3 XMB Image and Audio Converter

art2xmb is a Python GUI tool that allows you to convert images and audio for use with the PlayStation 3's XrossMediaBar (XMB) interface. You can use this tool to prepare images and audio files to create custom themes and icons for your PS3.

## Features

- Load and process disc images.
- Load and process logo images.
- Load and process background images.
- Generate a preview of your custom theme.
- Load audio from YouTube links.
- Adjust audio volume.
- Process audio files.

## How to Use

1. **Load Disc Image**: Click the "Load Disc Image" button in the "Disc Image" section to select a disc image file. The tool will display the image and process it for PS3 use.

2. **Load Logo Image**: Click the "Load Logo Image" button in the "Logo Image" section to select a logo image file. The tool will display the image and process it for PS3 use.

3. **Load Background Image**: Click the "Load Background Image" button in the "Background Image" section to select a background image file. The tool will display the image and process it for PS3 use.

4. **Load All Images**: Click the "Load All Images" button in the "All Images" section to load and process all images at once.

5. **Generate Preview**: After loading images, click the "Generate Preview" button to create a preview of your custom theme. The preview will be displayed in the "All Images" section.

6. **Process All Images**: After loading images, click the "Process All Images" button to process all loaded images for PS3 use.

7. **Load Audio File**: Enter a YouTube link in the "YouTube Link" text box and click the "Load Audio File" button to load audio from YouTube.

8. **Play/Pause Audio**: Use the "Play" and "Pause" buttons to control audio playback.

9. **Adjust Volume**: Use the volume slider to adjust the audio volume.

10. **Process Audio**: Click the "Process Audio" button to process the loaded audio.

11. **Exit**: Close the application by clicking the close button on the main window.

## Note

- This tool is a work in progress, and some features yet fully implemented.
- Make sure to save the processed images to your desired location after processing.
